# AgentBets

AI-native prediction markets on Solana, built for the Colosseum Agent Hackathon.

## Commands

```
/agentbets markets [query]    Search and list markets
/agentbets market <id>        Show detailed market information
/agentbets opportunities      Find +EV trading opportunities
/agentbets help               Show help
```
